(begin e1 e2 ...)
Evaluates expressions in order, returning the last.
(begin (print 1) (+ 2 3)) => 5
(cond (test result...) ... (else result...))
Multi-branch conditional; else always matches.
(cond ((< x 0) "neg") (else "non-neg"))
(quote expr) or 'expr
Returns expr unevaluated (data, not code).
'(+ 1 2) => (+ 1 2)
(quasiquote expr) or `expr
Like quote, but (unquote x) / ,x evaluates.
`(+ 1 ,(* 2 3)) => (+ 1 6)
(unquote expr) or ,expr
Evaluates expr inside a quasiquote template.
(defmacro (name params...) template)
Defines a macro: params bind to unevaluated arguments, the template builds code.
(defmacro (unless c a b) `(if ,c ,b ,a))
(try expr fallback)
Evaluates expr; on error evaluates fallback instead.
(try (/ 1 0) "fallback") => "fallback"
(time expr)
Evaluates expr and prints the wall-clock time taken.
(trace fn-name)
Toggles call/return tracing for a user-defined function.
(print a b ...)
Writes arguments separated by spaces, then a newline; strings print raw.
(print "x =" 42)
(read)
Reads one line from input and parses it as an expression; eof at end of input.
(define x (read))
(load "file.lspm")
Evaluates a file in the current environment; returns the statement count.
(args)
Command-line arguments after the script path, as a list of strings.
(exit) or (exit code)
Terminates the process with an optional status code.
(random) or (random n)
Uniform random float in [0,1), or integer in [0,n).
(random 6) => 0..5
(random-seed n)
Seeds the PRNG for reproducible sequences.
(sort lst)
Ascending sort of numbers, rationals, and strings.
(sort (list 3 1 2)) => (list 1 2 3)
(assoc key alist)
First (key value) pair matching key in a list of pairs, else 0.
(assoc 2 (list (list 1 "a") (list 2 "b"))) => (list 2 "b")
(unit name)
One SI or derived unit as a dimension-carrying quantity (m kg s A K mol cd, km cm mm g min h Hz N Pa J W C V).
(/ (* 100 (unit km)) (unit h)) => 27.78 m/s
(concat s1 s2 ...)
Concatenates strings; non-string arguments are rendered.
(concat "x = " (+ 1 2)) => "x = 3"
(substring s start end)
Characters [start, end) of a string.
(substring "hello world" 0 5) => "hello"
(string->number s)
Parses a number (or p/q rational) from a string.
(string->number "22/7") => 22/7
(number->string x)
Renders any expression to its printed form as a string.
(number->string 1/3) => "1/3"
(split s sep)
Splits a string on a separator, returning a list of strings.
(split "a,b" ",") => (list "a" "b")
(numer x)
Numerator of an exact rational (integers are n/1).
(numer 22/7) => 22
(denom x)
Denominator of an exact rational (integers are n/1).
(denom 22/7) => 7
(number? x)
True for numbers and exact rationals.
(integer? x)
True for integer-valued numbers.
(rational? x)
True for exact rationals and integers.
(symbol? x)
True for symbols.

(complex? x)

(apply f arg-list)
Calls f with the elements of arg-list as arguments.
(apply + '(1 2 3)) => 6
(error "message" ...)
Raises a user error, printing the message.

Errors when cond is false; the test runner counts failures.

(define name value) or (define (name params...) body)
Binds a value, or defines a function (captures free variables by value).
(define (square x) (* x x))
(lambda (params...) body)
Creates an anonymous function (closure).

(let ((var val) ...) body)
Local bindings with lexical scope.
(let ((a 3) (b 4)) (+ a b)) => 7
(letrec ((var val) ...) body)
Recursive local bindings (functions can call themselves).

(if condition then-expr [else-expr])
Conditional; stays symbolic when the condition is undecidable.
